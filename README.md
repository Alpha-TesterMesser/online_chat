# Simple Chat

Minimal chat app (Node.js + Socket.IO). Messages stored in memory (no DB).

## Run locally
1. Install Node 18+.
2. Clone repo and `cd simple-chat`.
3. Install:
   ```bash
   npm install
4. Start:
     npm start
Open http://localhost:3000 in your browser

That's all you need, so
Good Luck!
